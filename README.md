## Usage
The current version supports stdin and stdout.  
```bash
cat accesslog | at3 
```
then, Output formatted csv to stdout. 

### CSV format
* host
* remote user
* user
* time
* http method
* request uri
* http protocol
* http status code
* http referer
* user agent
